
# AI Driven Emotion Detection for Women's Safety

## Overview
This project implements a multimodal AI system that detects emotional distress using facial expressions and speech patterns to provide real-time emergency assistance.

## Features
- Facial Emotion Recognition (CNN - Integratable)
- Speech Emotion Recognition (LSTM - Integratable)
- Multimodal Fusion
- Emergency Alert Simulation

## Run Project
pip install -r requirements.txt
python main.py
